# 📝 To-Do List Application

A simple and user-friendly **To-Do List** application developed using **Python** as part of the **CodSoft Python Programming Internship**. This project helps users efficiently manage their daily tasks through a clean command-line interface.

---

## 📌 Project Overview

The To-Do List application allows users to organize their tasks by adding, viewing, updating, and deleting them. It is designed to improve productivity while demonstrating Python fundamentals such as functions, loops, lists, and conditional statements.

---

## ✨ Features

- ➕ Add new tasks
- 📋 View all tasks
- ✏️ Update existing tasks
- ❌ Delete completed or unwanted tasks
- 🔄 Menu-driven interface
- 💻 Easy-to-use command-line application

---

## 🛠️ Technologies Used

- **Python 3**
- Command Line Interface (CLI)

## 🚀 Getting Started

### Prerequisites

- Python 3.x installed on your system

### Installation

1. Clone this repository

```bash
git clone https://github.com/your-username/To-Do-List.git
```

2. Navigate to the project directory

```bash
cd To-Do-List
```

3. Run the application

```bash
python todo.py
```

---

## 💡 How It Works

1. Launch the application.
2. Select an option from the menu.
3. Add, view, update, or delete tasks.
4. Repeat until you choose to exit.
